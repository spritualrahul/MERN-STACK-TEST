## Installation


## Run Locally

Clone the project

```bash
  git clone https://github.com/spritualrahul/MERN-STACK-TEST.git
```

Go to the project directory

```bash
  cd server
  node index.js
```

Install dependencies

```bash
  cd client
  npm install
```

Start the server / frontend part

```bash
  npm start
```
